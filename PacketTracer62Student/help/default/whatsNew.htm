What's New

Users of previous versions of Packet Tracer will note a variety of new features in this version of Packet Tracer.

HTTP Server Improvements


  • Javascript and CSS support.
  • FTP server file import inside Packet Tracer.
  • HTTP and FTP linked file management support.

Physical Workspace


  • Network topology creation.
    • Device deployment.
    • Cabling of devices.
    • Wiring closet device deployment and cabling.
    • Unplugging and moving of cables.
  • Simulation mode support.

Other Changes


  • Stopped support for Windows XP.
  • Enhancements to EIGRP
  • Enhancements to OSPF
  • 3G/4G connectivity added to Smart Devices and Tablets

New Devices


  • 819 Router
  • Cell Tower
  • Central Office Server
  • Sniffer (deployable by segment)
